# CareNexus Lite 核心测试用例

所有执行结果以 `TEST_LOG.md` 为准，未执行不得标记通过。

| 编号 | 场景 | 预期 |
| --- | --- | --- |
| TC-AUTH-001 | 管理员/护工正确登录 | 返回Token并进入对应工作台 |
| TC-AUTH-002 | 错误密码或停用账号 | 拒绝登录且不泄露内部信息 |
| TC-AUTH-003 | 未登录/无权限访问 | 返回401/403统一JSON |
| TC-TRAIN-001 | 管理员新增编辑并发布资源 | 护工端可查看发布内容 |
| TC-TRAIN-002 | 下架已学习资源 | 护工不再新访问，历史记录保留 |
| TC-FILE-001 | 上传合法封面 | 安全存储并返回可访问URL |
| TC-FILE-002 | 上传空文件/非法类型 | 返回明确参数错误 |
| TC-LEARN-001 | 护工访问课程章节 | 更新访问次数、进度和最近时间 |
| TC-NOTE-001 | 保存图文笔记 | 笔记列表显示并可继续编辑 |
| TC-DISCUSS-001 | 发帖、回复、点赞 | 数据保存且计数正确 |
| TC-DISCUSS-002 | 删除他人内容 | 拒绝操作 |
| TC-ASSIGN-001 | 完成多题作业 | 自动评分、答题卡反馈，可重做 |
| TC-EXAM-001 | 未完成学习参加考试 | 拒绝进入 |
| TC-EXAM-002 | 提交单选/判断考试 | 自动评分并保存记录与错题 |
| TC-EXAM-003 | 超出考试次数 | 返回409且不新增记录 |
| TC-SCORE-001 | 查看课程与平均成绩 | 分数、通过状态和平均值正确 |
| TC-AI-001 | 基于已发布资料问答 | 返回答案和资料引用 |
| TC-AI-002 | 护工生成AI草稿 | 因无管理权限返回403 |
| TC-AI-003 | 管理员审核通过草稿 | 创建正式题库DRAFT题目 |
| TC-AI-004 | 重复审核 | 返回409 |
| TC-AI-005 | 请求医疗诊断/处方 | 不输出医疗决策 |
