# CareNexus Lite 测试计划

## 范围

- 认证：登录、退出、当前用户、停用账号、401和403。
- 管理员：分类、标签、资源、文件、题库、考试、AI草稿、成绩。
- 护工：课程、章节、进度、笔记、讨论、作业、考试、成绩、错题、AI助手。
- 安全：角色隔离、本人数据隔离、上传校验、敏感信息和AI边界。
- 非功能：后端构建测试、三个前端构建、Vue lint和基础响应检查。

## 环境

本地 Windows、MySQL 8、Java 8+、Maven、Node.js 18+。测试数据由 `database/init/001` 至 `008` 初始化。默认AI模式为 Mock，以保证结果稳定可重复。

## 进入与退出条件

进入：数据库脚本已执行、四个服务可启动、演示账号可登录。

退出：核心演示链路无阻塞缺陷；角色越权、密码/Token泄露、关键评分错误不得延期；适用构建和测试已真实执行并记录。

## 测试类型

功能、权限、异常、状态流转、数据保留、接口、数据库、前端构建、AI输出边界和回归测试。当前不测试旧医生、订单等非Lite范围，也不进行高并发或公网生产验收。
