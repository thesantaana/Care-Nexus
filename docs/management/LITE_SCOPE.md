# CareNexus Lite 范围基线

## 保留范围

- 角色：管理员、护工。
- 管理：分类、标签、培训资源、封面和文件、题库、考试、AI题目草稿、成绩。
- 学习：课程章节、学习记录、进度、笔记、讨论、作业、考试、错题和成绩。
- AI：资料问答、总结、学习建议、练习和题目草稿；默认 Mock，可切换 DeepSeek。
- 工程：Spring Boot 单体后端、Vue 管理端、Vue 护工端、React 门户端、MySQL 8。

## 明确排除

- 老人和家属账户与绑定。
- 医生、健康档案、预警、随访、干预和评估。
- 护理服务浏览、预约、订单、派单、评价和投诉。
- 支付、短信、HIS/EMR、可穿戴设备和医疗AI。
- 微服务、Redis、Docker、Nginx和CI/CD不是当前本地演示必需项。

## 演示闭环

管理员发布资料和考试 -> 护工登录学习 -> 记录笔记/讨论 -> 完成作业与考试 -> 系统评分并生成成绩/错题 -> 护工使用AI辅助 -> 管理员审核AI题目草稿。
