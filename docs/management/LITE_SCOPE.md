# CareNexus 轻量版范围

分支：`lite_develop`

## 保留范围

- 角色：管理员、护工/护理人员。
- 后端：认证与RBAC、培训资源、学习记录、题库考核、AI培训辅助、文件存储、操作审计。
- 前端：统一门户、管理员PC端、护工移动端。
- 数据库：21张用户权限、培训、考核、AI、文件和审计相关表。

管理员吸收原培训管理员和运营人员中与培训有关的职责，拥有培训资源查看与管理权限。护工拥有培训资源查看、学习记录和考核相关能力。

## 删除范围

- 运营人员、培训管理员、医生、健康管理人员、老人、家属角色及演示账号。
- 护理预约、地址、订单、分配、执行、评价和投诉模块。
- 医生老人授权、健康记录、预警、随访、干预和健康评估模块。
- 对应后端代码、数据库表、种子数据、测试、前端路由、菜单和宣传内容。

## 分支关系

`develop` 保留原完整业务代码，不受轻量版清理影响。轻量版后续修改只在 `lite_develop` 上推进。
