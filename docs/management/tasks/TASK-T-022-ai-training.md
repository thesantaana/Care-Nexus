## 实现范围

- 稳定 Mock 模式的培训资料问答、知识点总结和本人学习建议。
- 基于现有培训资料生成单选题或判断题草稿。
- 草稿分页、详情以及培训管理员审核通过或驳回。
- 审核通过后创建 DRAFT 正式题目；不自动发布、不自动加入考核。
- 所有输出带培训资源引用，且仅使用已校验的 TEXT 正文。

## 边界

- 不接真实模型，不需要 API Key，不访问互联网。
- 不解析 PDF、Word、PPT，不引入复杂 AI 框架、向量数据库或 Redis。
- 不读取医生健康数据，不输出诊断、处方、个体治疗方案或医疗决策。
- MySQL 完整 HTTP 联调和跨模块全量回归留到 T-023。
