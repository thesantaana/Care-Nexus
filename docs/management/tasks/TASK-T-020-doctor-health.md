## 实现范围

- 管理员、健康管理人员维护医生与老人授权关系并记录操作日志。
- 医生查询授权老人和健康档案，禁止访问未授权老人。
- 健康记录支持血压、血糖、心率、体温，至少填写一项指标。
- 基础阈值或人工操作形成健康预警；阈值仅用于风险提示，不构成医疗诊断。
- 健康预警按 `PENDING -> PROCESSING -> CLOSED` 流转。
- 随访、干预和健康评估先保存草稿，医生确认后形成正式记录。
- 医生端不实现 AI 诊断、AI 处方或自动医疗决策。

## 已确认规则

- 基础阈值使用配置项管理，首期覆盖血压、血糖、心率和体温。
- 健康记录必须包含老人、记录时间以及至少一项健康指标，备注可选。
- 已确认的随访、干预和评估不可直接修改，只能新增后续记录。
- 医生不能自行新增或取消授权，只能使用已有授权。
- 本轮执行聚焦自动化测试和 `mvn verify`；正式 MySQL 导入、完整 HTTP 联调和跨模块回归测试纳入 T-023。
