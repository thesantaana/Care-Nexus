# CareNexus Lite 文档索引

本目录以 `lite_develop` 当前可运行成果为准。历史日志保留实际过程，不代表当前产品仍包含其中规划过的全部模块。

| 目录 | 唯一用途 |
| --- | --- |
| `requirements/` | Lite 产品范围、角色、功能和业务流程 |
| `design/` | 当前模块化单体、前端结构、数据库和认证设计 |
| `api/` | 已实现 API 约定和接口清单 |
| `test/` | 当前测试策略、用例和真实执行记录 |
| `management/` | 项目状态、任务、范围、变更与问题 |
| `plans/` | Lite 后续迭代与配置管理 |
| `delivery/` | 演示、部署和交付说明 |
| `decisions/` | 已确认架构决策 |
| `logs/`、`meetings/` | 历史过程记录 |
| `workflows/` | 需求澄清流程 |

## 当前交付范围

管理员培训管理、护工课程学习、笔记讨论、作业考试、成绩错题和 AI 培训辅助。老人、家属、医生、健康管理、预约订单和支付不属于 Lite 交付。
