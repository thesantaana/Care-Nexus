# API 文档

适用基线：`lite_develop`，更新时间：2026-07-15。

本目录只维护 CareNexus 轻量版当前接口：认证、培训资源、学习、逐课考核、成绩、笔记、讨论、作业和 AI 培训辅助。

- `API设计规范.md`：统一响应、认证、权限、状态码、分页、文件和 AI 规则。
- `MVP接口清单.md`：按当前 Controller 路由整理的接口总表。

护理订单、老人家属和医生健康接口已从轻量版删除。历史文档中的 `/care/**`、`/doctor/**` 不得作为当前接入依据。

接口发生变化时应同步更新需求、数据库、测试和前端调用；没有实际 Controller 或 Service 实现的接口不得写入现行清单。
