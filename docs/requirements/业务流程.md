# 业务流程

项目名称：CareNexus 颐联  
版本：轻量版 2.0  
更新时间：2026-07-15

## 1. 登录与角色分流

```mermaid
flowchart TD
    A[用户打开统一门户] --> B[输入用户名和密码]
    B --> C{后端认证成功?}
    C -- 否 --> D[显示统一错误信息]
    C -- 是 --> E[返回 Token、角色和权限]
    E --> F{主角色}
    F -- ADMIN --> G[进入管理员工作台]
    F -- CAREGIVER --> H[进入护工学习端]
    G --> I[后续请求携带 Bearer Token]
    H --> I
    I --> J{Token与账号状态有效?}
    J -- 否 --> K[清除会话并重新登录]
    J -- 是 --> L[按权限访问功能]
```

## 2. 培训资源管理

```mermaid
flowchart TD
    A[管理员创建课程] --> B[选择文章/视频/PPT]
    B --> C[选择文本/本地文件/外部链接]
    C --> D[填写标题、摘要、封面、类别、标签和时长]
    D --> E[保存为 DRAFT]
    E --> F{需要修改?}
    F -- 是 --> D
    F -- 否 --> G[发布课程]
    G --> H[PUBLISHED]
    H --> I[护工可浏览和学习]
    H --> J{需要调整内容?}
    J -- 是 --> K[先下架]
    K --> L[OFFLINE]
    L --> D
    J -- 否 --> I
```

业务规则：

- 护工只能读取 `PUBLISHED` 资源。
- 已发布资源不能直接修改。
- 重复发布或重复下架返回冲突。
- 发布和下架写入操作日志。

## 3. 逐课学习与考核

```mermaid
flowchart TD
    A[护工浏览已发布课程] --> B[打开课程详情]
    B --> C[记录学习访问秒数]
    C --> D[累计课程与整体学习情况]
    D --> E[进入课程考核]
    E --> F[获取已发布考核与客观题]
    F --> G[提交答案]
    G --> H[系统自动评分]
    H --> I[保存考试记录和每题答案]
    I --> J{分数达到通过线?}
    J -- 是 --> K[课程状态 PASSED]
    J -- 否 --> L[课程状态 NOT_PASSED]
    K --> M[更新最高分与平均分]
    L --> M
    M --> N{全部课程通过?}
    N -- 是 --> O[整体培训 PASSED]
    N -- 否 --> P[整体培训保持未通过/学习中]
```

## 4. 题库与考核管理

```mermaid
flowchart TD
    A[管理员选择课程] --> B[创建单选题或判断题]
    B --> C[填写答案与解析]
    C --> D{单选题?}
    D -- 是 --> E[维护选项且只能有一个正确答案]
    D -- 否 --> F[保存布尔标准答案]
    E --> G[创建或修改课程考核]
    F --> G
    G --> H[关联题目与分值]
    H --> I[发布考核]
    I --> J[护工可参加]
```

每门课程只保留一份考核关系。系统不支持简答题和人工评分。

## 5. AI 学习辅助

```mermaid
flowchart TD
    A[用户选择文本培训资料] --> B[后端校验当前用户访问权限]
    B --> C{资料可访问且为TEXT?}
    C -- 否 --> D[拒绝请求]
    C -- 是 --> E{AI模式}
    E -- mock --> F[稳定Mock生成]
    E -- deepseek --> G[调用DeepSeek]
    F --> H[返回内容和来源引用]
    G --> H
    H --> I{资料足够?}
    I -- 否 --> J[明确说明资料不足]
    I -- 是 --> K[输出问答/总结/建议]
```

AI 不读取健康数据，也不输出诊断、处方、治疗方案或医疗决策。

## 6. AI 题目草稿审核

```mermaid
flowchart TD
    A[管理员选择文本资料和题型] --> B[生成题目草稿]
    B --> C[保存草稿与资料来源]
    C --> D[管理员查看草稿详情]
    D --> E{审核结果}
    E -- 驳回 --> F[草稿 REJECTED]
    E -- 通过 --> G[草稿 APPROVED]
    G --> H[创建正式题目 DRAFT]
    H --> I[不自动发布]
    I --> J[不自动加入考核]
```

重复审核返回冲突。草稿仅支持单选题和判断题。

## 7. 富文本课程笔记

```mermaid
flowchart TD
    A[护工进入课程笔记] --> B[按当前用户和课程查询]
    B --> C{已有笔记?}
    C -- 否 --> D[创建空白文档]
    C -- 是 --> E[加载标题与富文本内容]
    D --> F[编辑内容]
    E --> F
    F --> G{插入图片?}
    G -- 是 --> H[上传并返回图片URL]
    H --> F
    G -- 否 --> I[保存]
    I --> J[按用户+课程唯一关系写入]
```

任何用户都不能通过修改请求参数访问其他人的笔记。

## 8. 课程讨论

```mermaid
flowchart TD
    A[用户进入课程详情] --> B[查看讨论列表]
    B --> C[发布主题]
    B --> D[回复主题]
    D --> E[可指定父回复]
    B --> F[点赞/取消点赞主题]
    D --> G[点赞/取消点赞回复]
    C --> H{删除?}
    D --> H
    H -- 本人内容 --> I[允许删除]
    H -- 他人内容 --> J[拒绝]
```

## 9. 课后作业

```mermaid
flowchart TD
    A[护工查看课程作业] --> B[选择作业]
    B --> C[填写答案]
    C --> D[提交]
    D --> E{是否已有提交?}
    E -- 否 --> F[创建提交记录]
    E -- 是 --> G[按业务规则更新或拒绝重复]
    F --> H[返回提交状态]
    G --> H
```

## 10. 培训成绩查看

### 护工

1. 查询本人所有课程成绩。
2. 返回每课最高分、平均分、通过状态和尝试次数。
3. 汇总已通过课程数和整体培训状态。
4. 可按课程查看错题。

### 管理员

1. 查询护工成绩列表。
2. 查看各护工课程完成和通过情况。
3. 不直接修改原始考试记录。

## 11. 文件上传

```mermaid
flowchart TD
    A[上传封面/资料/笔记图片] --> B[校验认证与权限]
    B --> C[校验大小]
    C --> D[校验扩展名与MIME]
    D --> E{合法?}
    E -- 否 --> F[返回参数或安全错误]
    E -- 是 --> G[生成安全存储名]
    G --> H[保存到配置目录]
    H --> I[保存或返回访问地址]
```

## 12. 范围说明

本流程不包含护理预约、护理订单、家属绑定、医生健康管理和医疗 AI。历史业务流程中的相关图只代表完整版阶段，不属于轻量版当前验收主线。
