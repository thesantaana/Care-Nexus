# 设计文档

适用基线：`lite_develop`，更新时间：2026-07-15。

本目录说明 CareNexus 轻量版如何实现管理员与护工培训闭环。

- `系统架构设计.md`：前后端、后端模块、AI、文件和部署架构。
- `系统分析与设计模型.md`：上下文、类图、时序和状态模型。
- `原型设计.md`：统一门户、管理员端和护工端页面结构。
- `数据库设计.md`：001–008 最终 28 张表及关系。
- `认证设计说明.md`：JWT、RBAC、账号状态和前端会话。
- `工程骨架说明.md`：实际仓库、包结构、配置、启动和构建。

现行设计不包含护理订单、老人家属和医生健康管理。历史 36 表模型、care/doctor 包结构或待接入页面不属于当前设计基线。

设计变化必须能够追踪到需求、代码、数据库、API 和测试。重大技术选择通过 ADR 记录。
