# CareNexus Lite 工程说明

项目已不再是空骨架，当前包含可运行培训业务。

```text
backend/src/main/java/com/carenexus/
  auth/ training/ ai/ file/ audit/ common/
frontend/
  portal-web/ admin-web/ mobile-web/
database/
  init/ dict/ model/
scripts/
  start-lite.ps1 stop-lite.ps1
```

后端以 Maven 构建，三个前端各自维护依赖和 Vite 配置。门户负责品牌与入口，管理员端负责内容与考核管理，护工端负责课程学习。数据库通过编号SQL初始化，不依赖应用自动建表。

本地开发端口固定为 8080、5173、5174、5175；若端口占用，先执行 `scripts/stop-lite.ps1`。
