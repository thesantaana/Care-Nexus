文档状态：已审核，T-011封板

本文档记录 T-011 阶段的低保真页面原型。原型用于说明页面信息结构和主要入口，不代表已经实现完整业务功能。

## 1. PC首页

```mermaid
flowchart TB
    A["顶部：CareNexus / 当前用户 / 退出"]
    B["左侧菜单：培训管理 / 医生服务 / 综合管理 / 护工学习"]
    C["今日待办：待审核AI草稿、待分配订单、待处理预警"]
    D["数据概览：培训资源数、学习人数、订单数、健康预警数"]
    E["快捷入口：新增培训资源、分配订单、医生老人授权、查看日志"]
    A --> B
    A --> C
    C --> D
    D --> E
```

## 8. 当前Vue静态原型位置

- PC 首页和培训、医生、综合管理入口位于 `frontend/admin-web/`。
- 移动端服务首页、服务预约、订单执行和移动培训入口位于 `frontend/mobile-web/`。
- 当前页面仅用于展示信息结构和路由骨架，不包含真实接口联调或完整业务逻辑。
